3Alternative: Manual Script

If the bookmarklet doesn't work, you can run this script in the browser console:

async function getMemberList() { try { // Get the report data from the current page const response = await fetch('https://lcr.churchofjesuschrist.org/api/report/custom-reports/run-report/270dd333-769f-43a0-b73e-d27cc6d5d730?lang=eng'); const json = await response.json(); const rows = []; const header = []; // Extract headers for(const column of json.columns) { header.push(column.key); } rows.push(header); // Extract member data for(const member of json.members) { const row = []; for(const column of json.columns) { row.push(member[column.key] || ''); } rows.push(row); } // Convert to tab-separated format const tabData = rows.map(row => row.join('\t')).join('\n'); // Copy to clipboard await navigator.clipboard.writeText(tabData); alert('‚úÖ Data extracted and copied to clipboard!'); console.log('Report data:', tabData); return tabData; } catch (error) { console.error('Error extracting data:', error); alert('‚ùå Error extracting data. Check console for details.'); } } // Run the extraction getMemberList();

How It Works

The bookmarklet automatically:

  • Fetches the member report data from LCR
  • Converts it to the correct format (tab-separated)
  • Copies the data to your clipboard
  • Shows a success message
