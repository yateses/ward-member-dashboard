LCR Data Fetch Bookmarklet

Since popup blocking is preventing automatic data fetching, you can use this bookmarklet to fetch data directly from the LCR website.

How to Use:

Step 1: Drag this button to your bookmarks bar:
ðŸ“Š Fetch LCR Data
Step 2: Go to LCR and log in
Step 3: Click the bookmarklet in your bookmarks bar
Step 4: The data will be copied to your clipboard automatically
Step 5: Go back to your app and paste the data in the "Manual Paste" section

Alternative Method:

If the bookmarklet doesn't work, you can also:

  1. Open the browser console on LCR (F12)
  2. Copy and paste this code:
async function fetchLCRData() {
    const response = await fetch('https://lcr.churchofjesuschrist.org/api/report/custom-reports/run-report/33D30D27-9F76-A043-B73E-D27CC6D5D730?lang=eng', {
        credentials: 'include',
        headers: {
            'Accept': 'application/json',
            'Content-Type': 'application/json'
        }
    });
    
    const data = await response.json();
    console.log('Data:', data);
    
    // Convert to tab-separated format
    const headers = data.columns.map(col => col.key);
    const rows = [headers, ...data.members.map(member => 
        headers.map(header => member[header] || '').join('\t')
    )];
    const tabData = rows.join('\n');
    
    console.log('Tab-separated data:');
    console.log(tabData);
    
    return tabData;
}

fetchLCRData();

Note: This method works around popup blocking by running the fetch directly on the LCR website where you're already logged in.
